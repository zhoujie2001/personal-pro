# OpenClaw Workspace

This is an OpenClaw workspace configuration repository.

## Contents

- `AGENTS.md` - Workspace guidelines and agent behavior
- `SOUL.md` - Agent personality and core principles
- `USER.md` - User information and preferences
- `TOOLS.md` - Local tool configurations and notes
- `IDENTITY.md` - Agent identity definition
- `HEARTBEAT.md` - Periodic check tasks
- `BOOTSTRAP.md` - Initial setup instructions

## Purpose

This repository contains the configuration files for an OpenClaw AI assistant workspace. These files define:
- The assistant's personality and behavior
- User preferences and context
- Workspace organization and memory management
- Tool configurations and local settings

## Getting Started

1. Clone this repository to your OpenClaw workspace directory
2. OpenClaw will automatically load these configuration files
3. The assistant will adopt the personality and settings defined here

## Notes

- This is a configuration repository, not a code project
- Files are in Markdown format for easy editing
- The `.gitignore` file excludes temporary and sensitive data
